# Harry Chang Website

Based on the repo template by [timlrx](https://github.com/timlrx/tailwind-nextjs-starter-blog)!

## Project Structure

Here is an overview of the project's file structure:

- **app/**: Contains the Next.js App Router pages and layouts.
- **components/**: Reusable React components used throughout the site.
- **css/**: Global styles and Tailwind CSS configuration (v4).
- **data/**:
    - **blog/**: Markdown/MDX files for blog posts.
    - **authors/**: Author profiles.
    - **siteMetadata.js**: Configuration for site title, author, social links, etc.
    - **projectsData.ts**: Data for the projects page.
    - **headerNavLinks.ts**: Navigation links configuration.
- **layouts/**: Layout components for different page types (e.g., PostLayout, ListLayout).
- **public/**: Static assets like images and favicons.
- **scripts/**: Utility scripts for tasks like RSS generation.
- **contentlayer.config.ts**: Configuration for Contentlayer, which manages content transformation.
- **next.config.js**: Next.js configuration file.
- **package.json**: Project dependencies and scripts.
